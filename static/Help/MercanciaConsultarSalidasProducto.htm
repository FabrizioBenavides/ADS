Consultar salidas por producto de Transferencias

UbicaciónEsta página de cuarto nivel es parte del proceso de consulta detallada de salidas de mercancía. Para llegar a ella se sigue la trayectoria Mercancía > Salidas > Transferencias a otra sucursal > Consultar salidas por producto.
Descripción Esta página sirve para configurar el rango de fechas y el producto cuyas salidas por medio de transferencias a otras sucursales se quiere consultar.
Procedimiento

Seleccione primero qué mes desea consultar: el actual o el anterior.
Capture en seguida el código del artículo. Si no lo conoce, teclee algunas letras de su
descripción y haga clic en el icono en forma de lupa.
El sistema le mostrará una ventana con artículos que contienen las letras o números
capturados. Basta con hacer clic en el nombre de uno de estos productos para cerrar la
ventana y pasar a la forma de consulta el nombre de ese artículo.
Si el nombre del artículo situado junto a la lupa es el que desea consultar, está listo para
ejecutar la consulta haciendo clic en el botón Consultar. En caso contrario, repita el
procedimiento [3] hasta tener el artículo deseado


Si no hay salidas para la combinación de periodo y producto seleccionado, se le mostrará
una ventana con un mensaje informativo.
Si sí existen salidas, se desplegará una tabla reportando unidades que salieron para cada día del rango elegido.


Si no hay salidas para la combinación de periodo y producto seleccionado, se le mostrará
una ventana con un mensaje informativo.
Si sí existen salidas, se desplegará una tabla reportando unidades que salieron para cada día del rango elegido.
Oprima el botón Ver entradas para ver el detalle de entradas de ese mismo producto y
durante el mismo lapso seleccionado.
Oprima el botón Imprimir reporte para enviar a la impresora la lista de productos que
ve en pantalla.
