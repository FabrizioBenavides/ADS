Consultar máximos por producto

UbicaciónEsta es una página de cuarto nivel que da principio al proceso de consultar los máximos de un producto concreto. Para llegar a ella se sigue la trayectoria Mercancía > Inventarios >
Máximos por producto > Consultar máximos por producto.
Descripción Esta página sirve para definir un producto, consultar sus valores máximos y, sobre la base del desempeño histórico, poder ir a sugerir máximos.
Procedimiento
Inicie el proceso tecleando el código de un producto. Si no la conoce, escriba parte de su
nombre en el campo correspondiente y luego oprima el icono en forma de lupa. Esto le
mostrará una lista de productos, y bastará con que haga clic en uno de ellos para recuperarlo
y agregarlo a la forma.
Si el nombre y código del producto están correctos, oprima el botón Consultar. El
sistema le presentará una tabla con el inventario de ese producto y una tabla con las ventas
recientes del mismo producto.
Para consultar los máximos de otro producto, oprima el botón Limpiar.
Para imprimir las tablas con información del producto, oprima el botón Imprimir.
Para sugerir valores máximos para el producto actual, oprima el botón Sugerir máximo.
