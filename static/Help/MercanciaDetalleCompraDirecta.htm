Ayuda Mercancía

Detalle de compra directa

UbicaciónEsta página de quinto nivel es producto de la consulta de una compra directa. Para llegar a ella siguió la trayectoria Mercancía > Entradas > Compras directas > Consultar archivo de compras directa y después seleccionó una nota de entrada a consultar.
Descripción Esta página le muestra el detalle de una compra directa específica ocurrida en el pasado inmediato.
ProcedimientoPuesto que la página es una consulta de detalle, no ofrece opciones de control que afecten al sistema.

Si la nota de entrada contiene muchos artículos, no todos aparecerán en la página inicial.
Para navegar hasta ellos, use las flechas de paginación.
Para ver el detalle de otra nota de entrada, oprima el botón Otro detalle y el sistema lo
devolverá a la lista de notas.
Para imprimir la lista completa de artículos de esta nota de entrada, oprima el botón
Imprimir.
