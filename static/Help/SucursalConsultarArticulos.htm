Ayuda Sucursal

Consultar artículos

UbicaciónEsta página de tercer nivel es parte del proceso de consulta de catálogos de la sucursal. Para llegar a ella se sigue la trayectoria Sucursal > Catálogos > Consultar artículos.
Descripción Esta página sirve para configurar la consulta de una lista de artículos según diversos criterios.
Procedimiento
Para recuperar la lista completa de artículos, no haga nada. Por defecto está seleccionada la opción “Todos los artículos de la sucursal”.
Para recuperar la lista de artículos de un departamento, marque el botón de “Todos los artículos de un departamento” y luego use el selector para elegir un departamento deseado.
Para recuperar la lista de artículos de un rango de departamentos, marque el botón de “Todos los artículos de un rango de departamentos” y luego use los selectores para elegir el departamento inicial y el departamento final. Los artículos recuperados serán los comprendidos entre ambos departamentos.
Para recuperar todos los artículos de una categoría, marque el botón de “Todos los artículos de una categoría” y luego use el selector para elegir una categoría deseada.
Para recuperar un artículo concreto, marque el botón de “Un artículo concreto”; luego teclee el código del producto o escriba parte de su nombre en el campo “Código de producto” y oprima el icono en forma de lupa. Esto le mostrará una lista de productos, y bastará con que haga clic en el correcto para recuperarlo y agregarlo a la forma.
Oprima el botón “Consultar”. El sistema le mostrará la lista de productos que cumplen con el criterio especificado. Si eligió la opción de un artículo concreto, el sistema le mostrará directamente la ficha de ese producto.


Haga clic en la descripción de un producto para ver su ficha.
Para imprimir el listado de producto oprima el botón “Imprimir lista”.
