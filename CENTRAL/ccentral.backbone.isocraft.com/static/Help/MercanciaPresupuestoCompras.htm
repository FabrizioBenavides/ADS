Ayuda Mercancía

Consultar presupuesto de gastos

UbicaciónEsta página de cuarto nivel permite consultar el presupuesto de gastos de la sucursal para un mes concreto. Para llegar a ella sigue la trayectoria Mercancía > Entradas > Consultar gastos > Consultar presupuesto de gastos.
Descripción Esta página le permite ver el presupuesto histórico de un mes, o bien conocer el monto de las compras hechas en el mes en curso respecto al presupuesto asignado.
Procedimiento

El proceso se reduce a seleccionar el mes para el que se desea ver el presupuesto de gastos
de la sucursal.
Una vez elegido el mes, oprima el botón Consultar.
El sistema le mostrará el presupuesto asignado para el mes que seleccionó, así como el monto ejercido, tanto en términos absolutos (pesos) como relativos (porcentaje del presupuesto total).
Para imprimir el reporte, oprima el botón Imprimir.
Para ver el detalle de las compras hechas en ese mismo mes por departamento, oprima el
botón Ver compras por departamento.
