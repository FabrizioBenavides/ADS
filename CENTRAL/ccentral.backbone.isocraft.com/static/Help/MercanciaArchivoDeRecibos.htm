Ayuda Mercancía

Consultar archivo de recibos

UbicaciónEsta página de cuarto nivel abre el proceso para recuperar un recibo de mercancía de Transferencias. Para llegar a ella sigue la trayectoria Mercancía > Entradas > Transferencia de otra sucursal > Consultar archivo de recibos.
Descripción Esta página sirve para configurar una búsqueda en el registro histórico de recibos de mercancía provenientes de transferencias de otra sucursal, y para elegir entre los resultados cuál recibo consultar en detalle.
ProcedimientoEl sistema le ofrece tres opciones para recuperar un listado de recibos archivados. Puede ver todos los recibos del mes actual, todos los del mes anterior, o bien seleccionar un rango de fechas para ver todos los recibos de transferencias comprendidos en ese rango.

El primer paso es seleccionar una de estas opciones. Si eligió consultar por mes (actual o anterior), basta con que ahora oprima el botón Consultar.
Si eligió consultar por rango, debe capturar dos fechas límite: desde qué fecha hasta qué fecha consultar. Luego oprima el botón Consultar.
NOTA: Si pone una sola fecha, el sistema le mostrará los recibos de mercancía de ese día.
La búsqueda le mostrará una tabla con la lista de transferencias que cumplen con el criterio seleccionado, organizadas por número de folio. Para consultar el detalle de un recibo, haga clic en su número de folio.


Contenido relacionado Existe un proceso similar para el proceso de consulta de transferencias a otras sucursales, al cual se puede llegar vía Mercancía > Salidas > Transferencias a otra sucursal > Consultar archivo de envíos.
