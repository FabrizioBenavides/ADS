Consultar entradas por producto

UbicaciónEsta página de cuarto nivel abre el proceso para consultar las entradas de mercancía de un producto concreto. Para llegar a ella sigue la trayectoria Mercancía > Entradas > Recepción de Mercancía > Consultar entradas por producto.
Descripción Esta página sirve para definir el producto cuyas entradas desea consultar.
Procedimiento

Defina primero, haciendo clic en un botón, si desea consultar las entradas para el mes en curso o para el mes anterior.
Capture enseguida el código del artículo. Si no lo conoce, teclee algunas letras de su descripción y haga clic en el icono en forma de lupa.
El sistema le mostrará una ventana con artículos que contienen las letras o números capturados. Basta con hacer clic en el nombre de uno de estos productos para cerrar la
ventana y pasar a la forma de consulta el nombre de ese artículo.
Si el nombre del artículo situado junto a la lupa es el que desea consultar, está listo para
ejecutar la consulta haciendo clic en el botón Consultar. En caso contrario, repita el
procedimiento [3] hasta tener el artículo deseado.


Si no hay entradas para la combinación de período y producto seleccionado, se le mostrará una ventana con un mensaje informativo. Si sí existen entradas, se desplegará una tabla reportando unidades que entraron para cada día del rango elegido.
Una vez en el reporte, podrá oprimir el botón Limpiar para volver a iniciar el proceso de consulta de entradas.
Oprima el botón Ver salidas para ver el detalle de salidas de ese mismo producto y durante el mismo período seleccionado.
Oprima el botón Imprimir reporte para enviar a la impresora la lista de productos que ve en pantalla.


Contenido relacionado Existe una página equivalente que contiene las salidas por producto. Para llegar a ella siga la ruta Mercancía > Salidas > Transferencias a otra sucursal > Consultar salidas por producto.
