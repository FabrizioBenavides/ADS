Ayuda Mercancía

Consultar registro de compras directas

UbicaciónEsta página de cuarto nivel abre el proceso para capturar una compra directa. Para llegar a ella sigue la trayectoria Mercancía > Entradas > Compras directas > Consultar registro de compras directas.
Descripción Esta página sirve para configurar la consulta de registros de compras directas.
Procedimiento

Para iniciar el proceso de consulta de las compras directas, elija si desea consultar el mes actual o el mes anterior.
Elija ahora cómo ordenar los resultados: por nota de entrada, por número de proveedor o por fecha de recepción de la factura.
Oprima en seguida el botón Consultar.


  El resultado será una tabla mostrando las notas de entrada de compras directas que cumplan con los criterios marcados.
Para navegar entre el total de notas de entrada, utilice las flechas de navegación.
Para ver el detalle de una compra directa, haga clic en la lupa situada a la derecha de la nota que quiera consultar.
Para imprimir la lista de notas recuperadas, oprima el botón Imprimir consulta.
Para iniciar una nueva consulta, oprima el botón Otra consulta.

NOTA: Mensualmente se debe imprimir un reporte por Folio (nota de entrada) y archivarse junto con los originales de las facturas recibidas en un fólder mensual.
