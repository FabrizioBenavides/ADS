Capturar transferencia interna

UbicaciónEsta página de cuarto nivel es parte del proceso de registro de transferencias internas. Para llegar a ella se sigue la trayectoria Mercancía > Salidas > Transferencias internas > Capturar transferencia interna.
Descripción Esta página sirve para registrar en detalle los movimientos internos de artículos.
ProcedimientoEl procedimiento a seguir consta de dos grandes rubros: definir la transferencia y luego definir los artículos comprendidos en ella.

Para empezar, elija el botón que corresponda al tipo de transferencia (Consumo ó Venta).
Use el calendario para seleccionar la fecha de la transferencia (por lo común, el día de hoy).
Utilice el selector para elegir el departamento que transfiere la mercancía.
Si la transferencia es de Consumo, el sistema No solicitará departamento solicitante.
Si aplica, use el selector para elegir el departamento al que se transfiere la mercancía.
Elija la cuenta de gastos a la que se aplicará el cargo. (32: Mantenimiento de equipo y edificio y 33: Papelería y artículos de escritorio).

Cuando se trate de una transferencia interna para venta, no se especifica la cuenta.

Para continuar el proceso, registre uno por uno los productos a transferir. Para cada uno de ellos tiene que registrar el código de producto y las unidades transferidas.
Para buscar un producto, teclee su código o escriba parte de su nombre en el campo correspondiente y luego oprima el icono en forma de lupa. Esto le mostrará una lista de productos, y bastará con que haga clic en uno de ellos para recuperarlo y agregarlo a la forma.
Si el nombre y código del producto están correctos, teclee en el campo Cantidad el número de unidades a transferir.
Oprima ahora el botón Agregar.


Repita los pasos [6] a [8] para cada producto a transferir. Esto iniciará la construcción de una tabla a la que irá agregando productos.
Si se equivoca en la adición de un producto, podrá borrarlo de la lista haciendo clic en la opción Borrar (icono de bote de basura).
Cuando termine de registrar los artículos transferidos y sus cantidades, sume las cantidades y anote el resultado en el campo Cifra de control.
Si desea cancelar los datos capturados, puede oprimir el botón Cancelar, pero debe hacerlo antes de registrar la transferencia.
Para terminar el proceso, oprima el botón Registrar.


Contenido relacionado Un proceso muy similar es el que se realiza para capturar transferencias manuales, sólo que en aquel caso es para transferir a otra sucursal, y para ello se sigue la ruta Mercancía > Salidas > Transferencias a otra sucursal > Capturar transferencia manual.
