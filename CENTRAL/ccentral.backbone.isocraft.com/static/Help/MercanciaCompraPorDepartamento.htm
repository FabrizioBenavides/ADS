Ayuda Mercancía

Compras por departamento

UbicaciónEsta página de quinto nivel permite consultar los gastos de su sucursal a nivel departamento. Para llegar a ella sigue la trayectoria Mercancía > Entradas > Consultar gastos > Compras por departamento.
Descripción Esta página le permite ver el presupuesto histórico de un mes concreto, detallado por
departamento.
Procedimiento

El proceso se reduce a seleccionar el mes para el que se desea ver el presupuesto de gastos por departamento.
Una vez elegido el mes, oprima el botón Consultar.

El sistema le mostrará las compras realizadas por cada departamento de la sucursal para el
mes que seleccionó, incluyendo las facturas que amparan dichas compras y el porcentaje que
las compras de un departamento representan respecto al total.
Si las compras son demasiadas para verlas en una sola pantalla, puede recorrer las compras usando las flechas de navegación disponibles en la tabla.
Para imprimir el reporte, oprima el botón Imprimir.
Para ver el presupuesto ejercido en ese mismo mes a nivel sucursal, oprima el botón Ver
presupuesto de gastos.
