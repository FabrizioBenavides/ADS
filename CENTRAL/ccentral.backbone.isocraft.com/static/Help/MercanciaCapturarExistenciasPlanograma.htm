Capturar existencias de un planograma

UbicaciónEsta es una página de cuarto nivel que continúa el proceso de captura física de existencias de un inventario rotativo. Para llegar a ella se siguió la trayectoria Mercancía > Inventarios > Inventarios rotativos > Capturar inventario rotativo y posteriormente se seleccionó un planograma, en el que se capturarán las existencias.
Descripción Esta página sirve para registrar las cantidades físicas de los artículos que están inventariados en un planograma específico.
ProcedimientoPara capturar un planograma, es necesario tener a la mano el reporte con el que se levantó las existencias del inventario rotativo.
Inicie el proceso registrando junto a cada código de producto las existencias inventariadas según el reporte.
Continúe capturando existencias hasta completar los artículos contenidos en la página.
Sume las cantidades registradas de esa página y anote la cantidad total en el campo Cifra de control.
Registre la hora en la que se tomó el inventario, No es la hora de captura.
Si existe, haga clic en la liga llamada Página siguiente. Si la cifra de control es correcta, esto lo llevará a la siguiente pantalla de captura del mismo planograma.
Repita la captura y validación para cada página de artículos, hasta terminar de registrar todo lo anotado en el reporte físico.
Para terminar el proceso, oprima el botón Salvar captura.
Para iniciar la captura de datos de otro planograma, oprima el botón Otro planograma.
