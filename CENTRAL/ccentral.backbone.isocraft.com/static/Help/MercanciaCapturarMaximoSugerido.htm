Ayuda Mercancía

Capturar sugerencia de máximos

UbicaciónEsta es una página de cuarto nivel que permite sugerir valores máximos para un producto
concreto. Para llegar a ella se sigue la trayectoria Mercancía > Inventarios > Máximos por producto > Capturar sugerencia de máximos.
Descripción Esta página sirve para sugerir modificaciones a los valores máximos para productos específicos.
Procedimiento
Para iniciar el proceso, introduzca el código del producto cuyo máximo desea sugerir. Si no la conoce, escriba parte de su nombre en el campo correspondiente y luego oprima el icono en forma de lupa. Esto le devolverá una lista de productos, y bastará con que haga clic en uno de ellos para recuperarlo y agregarlo a la forma.
Si el nombre y código del producto están correctos el sistema le presentará datos de inventario y ventas recientes para el producto señalado.
Sobre la base de estos datos, escriba en el campo Máximo sugerido la cantidad que recomienda tener en inventario.
Oprima el botón Sugerir máximo. El sistema empezará a construir una tabla de sugerencias con el producto y su máximo sugerido.
Repita los pasos [1] a [4] hasta que haya terminado de capturar las sugerencias de máximos.
Si quiere retirar un producto de la lista, puede hacer clic en su opción de Borrar (icono de bote de basura).
